在文件中插入

```jsp
<%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>

<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
```





例如:

```xml
<html>
<body>
<h2>雷猴啊!</h2>
<%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">

<input type="button" value="查询" onclick="findall"/>
</body>
</html>
```
